배운 태그들

  • <!DOCTYPE>

    • 이 문서가 HTML로 작성되었음을 알려주는 태그(?). HTML로 작성된 문서라면 맨 위에 적어주는 것이 국룰

  • <html>

    • <!DOCTYPE>를 제외한 모든 태그를 감싸는 태그. lang attribute는 이 <html> 태그 안에 들어가야 한다. 검색 엔진과 브라우저가 해당 페이지의 언어를 파악하는데 사용하기 때문.

  • <head>

    • 브라우저의 화면에 보이지 않는 것들을 작성하는 곳. 아직 여기에 대해선 많이 배우지 않음.

  • <title>

    • 브라우저의 탭에 표시될 내용을 작성할 수 있는 태그.

  • <meta>

    • 메타데이터를 작성하는 태그. 아직 내가 아는건 여기에 charset 속성이 들어간다는 정도밖에 안 된다. 검색해보니 컨텐츠의 이름, 컨텐츠 내용 정도를 넣을 수 있는 듯.

  • <body>

    • 화면에 나올 요소들을 작성하는 부분

  • <h1>

    • 제목을 적을 때 사용

  • <ol>

    • ordered list의 약자. 하위 태그로 <li>를 사용.

  • <ul>

    • unordered list의 약자. 하위 태그로 <li>를 사용.

  • <a>

    • 링크를 거는 태그. href attribute를 사용하여 링크할 주소를 입력한다. target="_blank" 속성을 사용하면 새 창에서 열기가 적용된다.

  • <img>

    • 이미지를 입력하는 빈 태그. 닫는 태그가 없다. 드라이브의 이미지를 불러올 때는 source의 약자인 src attribute를 사용한다. width 속성을 사용하면 크기를 지정할 수 있다.
